# Story 1.8: Mix Recording & Social Sharing

## Status

## Story

**As a** DJ,
**I want** to record my mix and create shareable clips,
**so that** I can promote my skills on social media.

## Acceptance Criteria

1. Record button starts/stops mix capture
2. MediaRecorder API captures master output
3. Visual recording indicator with elapsed time
4. Maximum recording length: 10 minutes (storage limit)
5. Generate clips: 15s, 30s, 60s (Instagram), up to 3min (TikTok)
6. Waveform visualization of recorded mix
7. Trim controls for selecting highlight moments
8. Export formats: MP3 (audio), MP4 (with visualizer)
9. Share buttons for major platforms
10. Local download option

## Integration Verification

- IV1: Recording doesn't impact playback performance
- IV2: Recorded files properly cleaned from memory
- IV3: Share functionality uses existing meta tags

### MediaRecorder Configuration
```typescript
const setupRecording = async () => {
  // Get master output from audio context
  const audioContext = getAudioContext();
  const destination = audioContext.createMediaStreamDestination();
  
  // Connect master output to recorder
  masterGainNode.connect(destination);
  
  // Configure MediaRecorder
  const mediaRecorder = new MediaRecorder(destination.stream, {
    mimeType: 'audio/webm;codecs=opus',
    audioBitsPerSecond: 192000
  });
  
  // Handle data chunks
  const chunks: Blob[] = [];
  mediaRecorder.ondataavailable = (e) => {
    if (e.data.size > 0) {
      chunks.push(e.data);
    }
  };
  
  // Start recording with 1-second chunks
  mediaRecorder.start(1000);
  
  return { mediaRecorder, chunks };
};
```

### Clip Generation Algorithm
```typescript
const generateClip = async (
  recording: Blob,
  startTime: number,
  duration: number
): Promise<Blob> => {
  // Load recording into AudioBuffer
  const arrayBuffer = await recording.arrayBuffer();
  const audioBuffer = await audioContext.decodeAudioData(arrayBuffer);
  
  // Extract clip segment
  const sampleRate = audioBuffer.sampleRate;
  const startSample = startTime * sampleRate;
  const endSample = (startTime + duration) * sampleRate;
  
  // Create new buffer for clip
  const clipBuffer = audioContext.createBuffer(
    audioBuffer.numberOfChannels,
    endSample - startSample,
    sampleRate
  );
  
  // Copy samples
  for (let channel = 0; channel < audioBuffer.numberOfChannels; channel++) {
    const sourceData = audioBuffer.getChannelData(channel);
    const targetData = clipBuffer.getChannelData(channel);
    targetData.set(sourceData.slice(startSample, endSample));
  }
  
  // Encode to MP3
  return encodeToMP3(clipBuffer);
};
```

const detectHighlights = (audioBuffer: AudioBuffer): Highlight[] => {
  const highlights: Highlight[] = [];

  // Detect buildups (gradual amplitude increase)

### Video Generation with Canvas
```typescript
const generateVideo = async (
  audioClip: Blob,
  duration: number
): Promise<Blob> => {
  const canvas = document.createElement('canvas');
  canvas.width = 1080;
  canvas.height = 1920; // Vertical for social
  const ctx = canvas.getContext('2d');
  
  // Setup MediaRecorder for video
  const stream = canvas.captureStream(30); // 30fps
  const audioStream = await createAudioStream(audioClip);
  
  // Combine video and audio streams
  const combinedStream = new MediaStream([
    ...stream.getVideoTracks(),
    ...audioStream.getAudioTracks()
  ]);
  
  const recorder = new MediaRecorder(combinedStream, {
    mimeType: 'video/mp4',
    videoBitsPerSecond: 5000000 // 5Mbps
  });
  
  // Animate visualizer on canvas
  animateVisualizer(ctx, audioBuffer, duration);
  
  return recordStream(recorder, duration);
};
```

### Memory Management
```typescript
class RecordingManager {
  private recordings: Map<string, Blob> = new Map();
  private maxRecordings = 3;
  private maxSizePerRecording = 100 * 1024 * 1024; // 100MB
  
  addRecording(id: string, blob: Blob) {
    // Check size limit
    if (blob.size > this.maxSizePerRecording) {
      throw new Error('Recording too large');
    }
    
    // Remove oldest if at limit
    if (this.recordings.size >= this.maxRecordings) {
      const oldest = this.recordings.keys().next().value;
      this.deleteRecording(oldest);
    }
    
    this.recordings.set(id, blob);
  }
  
  deleteRecording(id: string) {
    const blob = this.recordings.get(id);
    if (blob) {
      // Release blob memory
      URL.revokeObjectURL(URL.createObjectURL(blob));
      this.recordings.delete(id);
    }
  }
  
  cleanup() {
    this.recordings.forEach((_, id) => this.deleteRecording(id));
  }
}
```
