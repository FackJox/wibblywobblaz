# Story 1.6: BPM Detection & Glitch Effect Integration

## Status
Draft

## Story

**As a** user,
**I want** the glitch effects to pulse in sync with the music's tempo,
**so that** the visual experience matches the energy of the music.

## Acceptance Criteria

1. Essentia.js WASM loaded and initialized
2. BPM detection for local MP3s (±5 BPM accuracy)
3. BPM estimation for streaming tracks via metadata/API
4. Glitch intensity mapped: 60-90 BPM (subtle), 90-120 (medium), 120-140 (intense), 140+ (maximum)
5. Glitch triggers on: beat hits, drops, crossfader moves, track starts
6. Existing glitch effects enhanced, not replaced
7. Performance maintains 60fps with effects active
8. Manual BPM adjustment option if detection fails
9. Visual BPM confidence indicator

## Integration Verification

- IV1: Existing glitch effect triggers still function
- IV2: No performance regression in story-3.1 effects
- IV3: BPM data properly cleaned up on track change

## Dev Notes

### Essentia.js Configuration
```typescript
// WASM initialization
import { EssentiaWASM } from 'essentia.js';

const initBPMDetector = async () => {
  const essentia = new EssentiaWASM();
  await essentia.init({
    wasmPath: '/dj/wasm/essentia-wasm.wasm',
    workerPath: '/dj/wasm/essentia-worker.js'
  });
  return essentia;
};
```

### BPM Detection Algorithm
```typescript
// Using RhythmExtractor2013 for accuracy
const detectBPM = (audioBuffer: AudioBuffer): BPMResult => {
  const rhythmExtractor = essentia.RhythmExtractor2013({
    frameSize: 2048,
    hopSize: 1024,
    sampleRate: audioBuffer.sampleRate
  });
  
  const result = rhythmExtractor(audioBuffer);
  return {
    bpm: result.bpm,
    confidence: result.confidence,
    beats: result.beats
  };
};
```

### Glitch Intensity Mapping
```typescript
interface GlitchIntensity {
  level: 'subtle' | 'medium' | 'intense' | 'maximum';
  color: string;
  effects: {
    distortion: number;
    rgbSplit: number;
    noise: number;
    flicker: number;
  };
}

const mapBPMToIntensity = (bpm: number): GlitchIntensity => {
  if (bpm < 90) return SUBTLE_GLITCH;    // Chill
  if (bpm < 120) return MEDIUM_GLITCH;   // House
  if (bpm < 140) return INTENSE_GLITCH;  // Techno
  return MAXIMUM_GLITCH;                 // Drum & Bass
};
```

### Integration with Story 3.1 Glitch
```typescript
// Extend existing glitch system
import { useGlitchEffect } from '@/hooks/use-glitch-effect'; // story-3.1

export const useGlitchSync = (bpm: number) => {
  const { triggerGlitch, setIntensity } = useGlitchEffect();
  
  useEffect(() => {
    const intensity = mapBPMToIntensity(bpm);
    setIntensity(intensity.level);
    
    // Trigger on beat
    const beatInterval = 60000 / bpm; // ms per beat
    const timer = setInterval(() => {
      triggerGlitch({ duration: 100, strength: intensity.effects });
    }, beatInterval);
    
    return () => clearInterval(timer);
  }, [bpm]);
};
```

### Streaming Source BPM Estimation
```typescript
// Heuristics for when real detection isn't possible
const estimateBPM = (track: TrackInfo): number => {
  // Try API metadata first
  if (track.bpm) return track.bpm;
  
  // Genre-based estimation
  const genreMap = {
    'hip hop': 85,
    'house': 128,
    'techno': 130,
    'drum and bass': 174,
    'pop': 120,
    'rock': 120,
    'ambient': 70
  };
  
  // Parse from title (e.g., "Song Name 128 BPM")
  const bpmMatch = track.title.match(/(\d{2,3})\s*bpm/i);
  if (bpmMatch) return parseInt(bpmMatch[1]);
  
  // Default to house tempo
  return 128;
};
```

### Performance Monitoring
```typescript
const monitorPerformance = () => {
  let lastTime = performance.now();
  let frames = 0;
  
  const checkFPS = () => {
    frames++;
    const now = performance.now();
    
    if (now >= lastTime + 1000) {
      const fps = frames * 1000 / (now - lastTime);
      
      if (fps < 30) {
        // Reduce glitch complexity
        setGlitchQuality('low');
      } else if (fps < 50) {
        setGlitchQuality('medium');
      } else {
        setGlitchQuality('high');
      }
      
      frames = 0;
      lastTime = now;
    }
    
    requestAnimationFrame(checkFPS);
  };
  
  checkFPS();
};
```
