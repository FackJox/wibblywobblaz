# Story 1.7: Auto-Mix Mode & Continuous Playback

## Status
Draft

## Story

**As a** party host,
**I want** the music to continue automatically when the DJ steps away,
**so that** the party never stops.

## Acceptance Criteria

1. Auto-mix toggle button on interface
2. Automatic track loading from queue when deck empty
3. Configurable transition type: cut, fade, or auto-detect
4. Transition timing based on track endings (fade out detection)
5. Crossfader animates during auto transitions
6. Manual override instantly disables auto-mix
7. Smart track selection based on BPM similarity (if available)
8. Minimum 2-hour continuous playback without intervention
9. Auto-mix status indicator

## Integration Verification

- IV1: Auto-mix doesn't interfere with manual control
- IV2: Smooth transition between auto and manual modes
- IV3: Queue management remains responsive during auto-mix

### Transition Detection Algorithm
```typescript
const detectTrackEnding = (audioBuffer: AudioBuffer): TransitionPoint => {
  const sampleRate = audioBuffer.sampleRate;
  const lastSeconds = 30;
  const samples = sampleRate * lastSeconds;
  const startIndex = audioBuffer.length - samples;
  
  // Analyze amplitude envelope
  const envelope = extractEnvelope(audioBuffer, startIndex);
  
  // Detect fade out
  const fadeOutPoint = detectFadeOut(envelope);
  if (fadeOutPoint) {
    return {
      time: fadeOutPoint,
      type: 'fade',
      confidence: 0.9
    };
  }
  
  // Detect beat drop off
  const beatDropOff = detectBeatDropOff(envelope);
  if (beatDropOff) {
    return {
      time: beatDropOff,
      type: 'beat_end',
      confidence: 0.7
    };
  }
  
  // Default to time-based
  return {
    time: audioBuffer.duration - 5, // 5 seconds before end
    type: 'timed',
    confidence: 0.5
  };
};
```

### Auto-Mix Service
```typescript
class AutoMixService {
  private state: AutoMixState = AutoMixState.DISABLED;
  private config: AutoMixConfig;
  private nextTrack: QueueItem | null = null;
  
  async startAutoMix() {
    this.state = AutoMixState.IDLE;
    this.monitorDecks();
  }
  
  private monitorDecks() {
    // Check deck states every 100ms
    setInterval(() => {
      if (this.shouldLoadNext()) {
        this.loadNextTrack();
      }
      if (this.shouldTransition()) {
        this.executeTransition();
      }
    }, 100);
  }
  
  private async executeTransition() {
    this.state = AutoMixState.TRANSITIONING;
    
    const duration = this.config.fadeDuration * 1000;
    const steps = 60; // 60fps animation
    const stepDuration = duration / steps;
    
    for (let i = 0; i <= steps; i++) {
      const position = i / steps;
      this.animateCrossfader(position);
      await sleep(stepDuration);
      
      // Check for manual override
      if (this.isManuallyOverridden()) {
        this.disable();
        return;
      }
    }
    
    this.state = AutoMixState.IDLE;
  }
}
```

### BPM Matching Logic
```typescript
const selectNextTrack = (
  currentBPM: number,
  queue: QueueItem[],
  config: AutoMixConfig
): QueueItem | null => {
  if (!config.bpmMatching || !currentBPM) {
    return queue[0]; // Just take next in queue
  }
  
  // Find tracks within ±10% BPM range
  const tolerance = currentBPM * 0.1;
  const minBPM = currentBPM - tolerance;
  const maxBPM = currentBPM + tolerance;
  
  const candidates = queue.filter(track => {
    const trackBPM = track.metadata?.bpm;
    return trackBPM && trackBPM >= minBPM && trackBPM <= maxBPM;
  });
  
  if (candidates.length === 0) {
    // No BPM match, take next anyway
    return queue[0];
  }
  
  // Add some randomness to avoid repetition
  const index = Math.floor(Math.random() * Math.min(3, candidates.length));
  return candidates[index];
};
```

const detectManualIntervention = (): boolean => {
  const triggers = [
    crossfaderMoved,
    deckControlTouched,
    trackLoadedManually,
    transportControlUsed,
    eqKnobTurned
  ];
  
  return triggers.some(trigger => trigger === true);
};
